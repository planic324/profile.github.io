  • setfont

    @mixin setfont($font-size) {...}

    example>
    .parent_test .testfont {
    @include setfont(12);
    }// result > .ja .parent_text .testfont{font-size:12px,font-family:'돋움',Dotum,Helvetica,'Apple SD Gothic Neo',sans-serif !important}
    Description
    12px기준으로 나눔폰트, 시스템폰트를 구분하여 지정해준다. 다국어 설정(html[lang=ja])이 되어 있으면 다국어 폰트로 적용한다.
    Parameters
    Name Description Type Default value
    $font-size 폰트사이즈 Length -
